UML-diagram
- PlantUML is used for creating the diagram:
- See syntax: https://plantuml.com/class-diagram
- Only Constructors and Fields are included in the UML-diagram.

Files
- diagram.plantuml 	= regularly generated and updated manually via IntelliJ.
- layout.plantuml 	= defines custom layout options. Includes "diagram.plantuml".
- UML_image.png 	= manually exported result as an image.
